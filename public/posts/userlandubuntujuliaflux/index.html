Instalasi Julia dan Flux pada Android

May 2, 2023

Dokumen ini memuat informasi tentang instalasi Flux dan Julia pada Android.

  1. Install UserLand melalui Google Play.
  2. Jalankan UserLand, pilih Ubuntu, pilih minimum terminal.
  3. Download Julia untuk ARM.
  4. Untar file Julia.
  5. Test installasi Julia.
  6. Add Flux.
  7. Test installasi Flux.
  8. Kerjakan tutorial Flux untuk pemula.
